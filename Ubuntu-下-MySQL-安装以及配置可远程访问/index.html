对于 Ubuntu 系统下安装 MySQL 后一般都不支持远程连接，只能通过 localhost 或者 127.0.0.1 进行连接，因此需要我们自己去配置，这里介绍两种方式：

创建一个新的访问用户，配置该用户可以远程访问。(推荐使用这种方式)
